Param

Method

Event

 

 

IsInTablev3.5.1.05 or higher

현재 커서가 테이블 안에 있는 경우를 체크합니다.

 

Syntax

    HRESULT IsInTable();

Parameters

없음

 

Return Values

TRUE

커서가 테이블 안에 있음

FALSE

커서가 테이블 안에 없음

 

Remarks

메소드는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

 

Sample Codes

VBScript

var object = new NamoSE("test");
object.editorStart();
var info = object.IsInTable();
if(info)
{
    alert("커서가 테이블 안에 있습니다.");
}
else
{
    alert("커서가 테이블 안에 없습니다.");
}
